Интернационализация приложения
Класс java.util.Locale позволяет  учесть  особенности  региональных  пред-ставлений алфавита, символов, чисел, дат и проч. Автоматически виртуальная 
машина использует текущие региональные установки операционной системы, 
но при необходимости их можно изменять. Для некоторых стран региональные 
параметры  устанавливаются  с  помощью  констант,  например: Locale.US, 
Locale.FRANCE.  Для всех  остальных  объект Locale нужно  создавать  с по-мощью конструктора, например:
Locale rus = newLocale("ru", "RU");
